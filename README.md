# LofiPad - Advanced Offline Notepad

A beautiful, distraction-free notepad for Windows. Works completely offline.

## Features

✓ **Offline-first** - Your notes stay on your computer  
✓ **Rich formatting** - Bold, italic, underline, strikethrough  
✓ **Dark mode** - Easy on the eyes  
✓ **Custom appearance** - Change fonts and colors  
✓ **Auto-save** - Your changes are saved automatically  
✓ **No internet needed** - Works completely offline  

## Download & Install

### Step 1: Install Visual C++ Runtime (one-time setup)

If you get an error about "ffmpeg.dll", download and install this first:

**[Visual C++ Redistributables 2022 (64-bit)](https://aka.ms/vs/17/release/vc_redist.x64.exe)**

1. Click the link above
2. Run the installer
3. Click "Install"
4. Restart your computer

### Step 2: Download LofiPad

[Download LofiPad.exe from our releases page](https://github.com/sayhitosuman/lofipad/releases)

1. Click the download button
2. Double-click the downloaded file
3. Start typing!

**That's it!** No installation needed.

## How to Use

- **Open LofiPad.exe** - App starts immediately
- **Type your notes** - Start writing
- **Format text** - Use the toolbar buttons or keyboard shortcuts
- **Change appearance** - Click the settings gear icon
- **Notes autosave** - Your work is always saved

## Keyboard Shortcuts

- `Ctrl + B` - Bold
- `Ctrl + I` - Italic
- `Ctrl + U` - Underline
- `Ctrl + Q` - Quit app

## Troubleshooting

**"ffmpeg.dll not found" error?**  
→ Install Visual C++ Redistributables (see Step 1 above)

**App won't start?**  
→ Make sure you installed Visual C++ runtime and restarted your computer

**Need help?**  
→ Visit: https://github.com/sayhitosuman/lofipad/issues

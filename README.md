# LofiPad - Advanced Offline Notepad

A beautiful, distraction-free notepad for your browser. Works completely offline.

## Features

✓ **Offline-first** - Your notes stay in your browser's local storage  
✓ **Rich formatting** - Bold, italic, underline, strikethrough  
✓ **Dark mode** - Easy on the eyes  
✓ **Custom appearance** - Change fonts and colors  
✓ **Auto-save** - Your changes are saved automatically  
✓ **No internet needed** - Works completely offline  

## Getting Started

1. Visit the hosted version or run it locally
2. Start typing!

## Keyboard Shortcuts

- `Ctrl + B` - Bold
- `Ctrl + I` - Italic
- `Ctrl + U` - Underline
- `Ctrl + ,` - Settings
- `Ctrl + F` - Find & Replace
- `Ctrl + \` - Toggle Sidebar

## About

LofiPad is built with React. It's lightweight, fast, and completely free to use.
